# **App Name**: JASTIP SIAU

## Core Features:

- Secure User Authentication: Allows members, couriers, and administrators to register and log in securely. Includes robust error handling, password recovery, and email/phone verification for a seamless entry point.
- User & Profile Management: Users can view and edit their personal profiles, including name, WhatsApp number, email, and address with an optional map location. Admins can update user roles (member/courier) and manage all user details.
- Courier Status & Order Acceptance: Couriers can set their online/offline availability. Members can browse and select available couriers for their orders. Couriers receive new order notifications and can accept or decline assignments via the app.
- Order Creation & Detail Input: Members can create new proxy shopping (Jastip) orders, providing product details and quantities. The courier assigned to the order will input the final product price and relevant service fees, which are stored for tracking.
- WhatsApp Notification Tool: Automated system to send real-time transactional notifications (e.g., order confirmations, delivery updates, courier status changes) to members and couriers via their registered WhatsApp numbers.
- Admin Management Dashboard: A comprehensive dashboard for administrators to monitor user activity, manage courier availability, review and modify orders, oversee financial transactions, and set overall application parameters and service fees.
- Dynamic Service Fee Suggester (AI Tool): An AI-powered tool available to admins that suggests optimal service fees for specific delivery routes, considering factors like distance, potential traffic, and other configurable parameters to assist in pricing decisions.

## Style Guidelines:

- Light color scheme, symbolizing efficiency and clarity. Primary brand color is a trustworthy, professional blue (#1768B3). This strong hue anchors interactive elements and key headings.
- The background color is a very light blue-gray (#F3F7FB), creating a clean and open canvas that reduces eye strain.
- An energetic accent color, a bright aqua (#66CCE0), is used for highlights, calls-to-action, and to draw attention to important information like new notifications or active statuses.
- Headline and body font: 'Inter', a neutral sans-serif that ensures high readability across all information-rich interfaces, suitable for both short labels and detailed order descriptions.
- Clean, line-based vector icons for navigation and actions, designed for intuitive recognition and consistent with a modern, service-oriented application aesthetic.
- Responsive, mobile-first layout prioritizing essential information and quick access to order management and user actions on small screens, adapting smoothly to larger displays.
- Subtle, purpose-driven animations to provide visual feedback for user interactions, such as form submissions, order status updates, and view transitions, enhancing perceived responsiveness.
